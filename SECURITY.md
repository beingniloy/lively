# Security Policy

## Supported Versions

Currently, only the latest version of Lively Wallpaper is supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅         |

## Reporting a Vulnerability

We take the security of our users seriously. If you discover a security vulnerability within Lively Wallpaper, please report it via GitHub Issues or by contacting the maintainers directly.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
