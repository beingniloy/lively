# Contributing to Lively Wallpaper

First off, thank you for considering contributing to Lively! It's people like you that make Lively such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Use the **GitHub Issue Tracker**.
- Describe the bug in detail and provide steps to reproduce.
- Include information about your device and Android version.

### Suggesting Enhancements
- Open a new issue with the "enhancement" label.
- Explain why this feature would be useful to most users.

### Pull Requests
- Follow the existing code style (Kotlin with Jetpack Compose).
- Ensure the project builds successfully before submitting.
- Write clear, concise commit messages.

## Code Style
- Use 4 spaces for indentation.
- Follow Material 3 design guidelines for UI components.
- Keep Composables small and reusable.
