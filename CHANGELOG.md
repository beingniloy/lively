# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-29

### Added
- Initial production release.
- Video wallpaper engine using ExoPlayer.
- Wallpaper Editor with Fit, Fill, and Center Crop modes.
- Pinch-to-zoom and position adjustments.
- Library and Recent wallpapers tracking.
- Favorites system for quick access.
- Material 3 Design with Dynamic Color support.
- Dark/Light/System theme options.
- Battery optimization settings.
- GitHub Actions for automated builds and releases.
